## Environment Variables
Create a `.env` file in the root directory.
Example:
```env
PORT=3000
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/payments_db
REDIS_HOST=localhost
REDIS_PORT=6379
```
## Installation
Install dependencies:
```bash
npm install
```
## Build Project
```bash
npm run build
```

## Run Development Server
```bash
npm run dev
```
## Run Tests
```bash
npm run test
```
